A console-based **Supermarket Management System** built in C++.  
The application simulates real-world supermarket operations with an Admin management panel and a Point-of-Sale (POS) system for employees.

This project demonstrates strong fundamentals in **Object-Oriented Programming, file handling, authentication logic, and inventory management systems**.

---

## 📌 Key Features

### 👨‍💼 Admin Panel
- Secure admin login
- Add new inventory items
- Add new employees
- Persistent storage of accounts and items

### 🧾 Employee POS System
- Employee authentication
- Item selection and billing
- Automatic stock deduction
- Payment handling with change calculation
- Automatic stock reordering when quantity ≤ 50

---

## 🏗️ Technical Highlights

- Object-Oriented Design (Classes & Modular Structure)
- File-Based Data Persistence (`items.txt`, `admin.txt`, `employees.txt`)
- XOR-based credential obfuscation (for demonstration purposes)
- Inventory auto-replenishment logic
- STL usage (`vector`, `string`, `algorithm`)
- Console-based interactive workflow
